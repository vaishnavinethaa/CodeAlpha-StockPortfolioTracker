# CodeAlpha-StockPortfolioTracker
Stock Portfolio Tracker using Python for CodeAlpha Internship# 📈 Stock Portfolio Tracker

## Description
This is a Stock Portfolio Tracker developed using Python as part of the CodeAlpha Python Programming Internship.

## Features
- Displays available stocks
- Accepts stock name and quantity
- Calculates total investment
- Supports multiple stock entries
- Displays the total portfolio value
